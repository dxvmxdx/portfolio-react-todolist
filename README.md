# react-todolist

리액트로 간단한 투두리스트 구현

## Description:

배열 자료구조에 데이터를 추가•업데이트•삭제하는 간단한 기능을 구현하였으며, 로컬저장소에 저장하여 새로고침하여도 기존 리스트가 초기화되지 않고 불러올 수 있게 구현하였습니다.

## Build with:

- ReactJS
- PostCSS

## Features:

- 로컬저장소에 데이터 저장
- 다크 모드 구현
- 체크박스 클릭 시 state 변경
- Todo 데이터 추가와 삭제
